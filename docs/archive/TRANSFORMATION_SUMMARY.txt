================================================================================
                         ROAST MODE STRATEGY
================================================================================

ROAST MODE uses system .orange color:
  • Primary action buttons: .orange instead of .accentColor
  • Toggles and controls: .orange tint
  • All while maintaining native iOS look

Code Pattern:
  @AppStorage("roastModeEnabled") private var roastMode = false
  private var accent: Color { roastMode ? .orange : .accentColor }
  Button("Action") { }.tint(accent)

Result: Seamless, native-feeling roast mode without custom styling
